RegexMatchResult Methods
A string represents a sequence of characters.

Note  If you do not see the get methods during scripting in Designer, it is because the option Show getters in Workflow Options dialog is not checked.
Methods that deal with surrogate pairs or processes characters in byte mode must be enabled first using the option Suggest also byte mode and surrogate functions in Workflow Options Dialog.

Scripting API | RegexMatchResult Methods
assign
assign(String aString) : Void
Assigns regex matching results to the specified RegexMatchResult variable. The same result can be achieved using the equal sign =.

Util.RegexMatchResult mr = mr.assign(Util.Regex.createMatchResult("abc","aaa abc bbb abc"));
Scripting API | RegexMatchResult Methods
GroupMatch
GroupMatch[String aGroupName] : String
getGroupMatch(String aGroupName) : String
Lists the regex search matches in the specified group.

String Pattern = "(?<g1>abc\d)";
String Input = "abc1 abc2 abc3 abc4 abc5 abc6 xyz";
Util.StringList g1;
Util.RegexMatchResult mr = Util.Regex.createMatchResult(Pattern,Input);
while (mr.match())
   {
   String Value = mr.GroupMatch["g1"]; 
   g1.add(Value);
   }

for (Int i=0;i<g1.Size;i++) {
      logMessage(g1[i]);   
}

SubMatch[Int aIndex] : String
getSubMatch(Int aIndex) : String

String Pattern = "abc\d";
String Input = "abc1 abc2 abc3 abc4 abc5 abc6 xyz";
Util.RegexMatchResult mr = Util.Regex.createMatchResult(Pattern,Input);
while (mr.match())
   {
   for (Int idx=0;idx<mr.SubMatchCount;idx++)
      {

      logMessage(mr.getSubMatch(idx));
      }
   }

Scripting API | RegexMatchResult Methods
SubMatchEndPosition
SubMatchEndPosition[Int aIndex] : Int
getSubMatchEndPosition(Int aIndex) : Int
Returns the ending position (i.e. the number of the last character) within the input string of the match with the specified index number. The index corresponds to the number of matches, i.e. the match with index number 2 is the second match in the input string.

String Pattern = "abc\d";
String Input = "abc1 abc2 abc3 abc4 abc5 abc6 xyz";
Util.RegexMatchResult mr = Util.Regex.createMatchResult(Pattern,Input);
while (mr.match())
   {
   for (Int idx=0;idx<mr.SubMatchCount;idx++)
      {

      logMessage(mr.getSubMatchEndPosition(idx).toString());
      }
   }
