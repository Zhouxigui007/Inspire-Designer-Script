TypeKeeper Methods
TypeKeeper methods allow scripts to inspect namespaces, objects and their members at runtime through a process known as reflection.

Note  If you do not see the get methods during scripting in Inspire Designer, it is because the option Show getters in the Workflow Options dialog is not checked.
Scripting API | TypeKeeper Methods
IsEnum
IsEnum : Bool
getIsEnum() : Bool
Returns true if the type is an enumeration.

TypeKeeper tk = do.typeof(File.ListParam);

if(tk.IsEnum)
   logMessage("File.ListParam is an Enum");
else
   logMessage("File.ListParam is not an Enum");

MemberParameterName
MemberParameterName[Int aMemberIndex, Int aParameterIndex] : String
getMemberParameterName(Int aMemberIndex, Int aParameterIndex) : String
Returns the parameter name for a specified member parameter.

TypeKeeper tk = do.typeof(WorkFlow);

Int count = tk.MemberCount;

Int parameters;

for(Int i=0; i<count; i++)
   {
   
   parameters = tk.MemberParameterCount[i];
   
   for(Int p=0; p<parameters; p++)
      {
         logMessage(tk.MemberName[i] + " has parameter " + String(tk.MemberParameterName[i,p]));
      }
   
   }
